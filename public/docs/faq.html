🖥️시스템 기본 정보

Q1: 농장 출입 관리 시스템이 정확히 무엇인가요?

A: 농장을 방문하는 모든 사람들의 출입을 디지털로 기록하고 관리하는 웹 기반 시스템입니다. QR코드를 통해 방문자가 직접 등록하거나, 관리자가 컴퓨터/스마트폰으로 등록할 수 있습니다.

주요 기능:

방문자 정보 등록 및 관리
QR코드 자동 등록 시스템
실시간 알림 서비스
방문 통계 및 분석
데이터 내보내기 (Excel)
Q2: 어떤 종류의 농장에서 사용할 수 있나요?

A: 모든 종류의 농장에서 사용 가능합니다.

적용 가능한 농장 유형:

  • 축산업: 한우, 젖소, 돼지, 닭, 오리 등
  • 농업: 벼, 채소, 과일, 곡물 등
  • 원예업: 화훼, 조경, 묘목 등
  • 특수 농업: 버섯, 인삼, 약초 등
  • 수산업: 양식장, 어류 사육장 등

🛠️기능 및 사용법

Q3: QR코드는 어떻게 만들고 설치하나요?

A: 시스템에서 자동으로 생성되며, 설치도 매우 간단합니다.

QR코드 생성 과정:

  1. 농장 등록 후 "QR코드 생성" 클릭
  2. 농장별 전용 QR코드 자동 생성
  3. "인쇄" 버튼으로 A4 용지에 출력
  4. 농장 입구나 안내판에 부착

설치 팁:

  • 눈높이에 설치 (120-150cm)
  • 비바람을 막는 투명 케이스 사용
  • 충분한 조명 확보
  • 주차장, 게이트, 사무실 입구 등 여러 곳에 설치
Q4: 방문자가 QR코드를 스캔하면 어떻게 되나요?

A: 방문자 등록 페이지로 자동 이동하여 간편하게 정보를 입력할 수 있습니다.

등록 과정:

  1. QR코드 스캔: 스마트폰 카메라로 스캔
  2. 농장 확인: 어떤 농장인지 자동 표시
  3. 정보 입력: 이름, 연락처, 주소, 방문 목적 등
  4. 개인정보 동의: 수집 및 이용 동의
  5. 등록 완료: 즉시 시스템에 저장
  6. 알림 발송: 농장 관리자에게 실시간 알림
Q5: 여러 명이 함께 방문하는 경우는 어떻게 하나요?

A: 여러 가지 방법으로 처리할 수 있습니다.

대표자 등록:

  • 대표자 1명이 등록 시 동행자 수 입력
  • "추가 메모"에 동행자 이름 기록
  • 간단하고 빠른 처리

개별 등록:

  • 각자 QR코드 스캔하여 개별 등록
  • 정확한 개인별 기록 유지
  • 방역 추적에 더 유리

관리자 일괄 등록:

  • 관리자가 컴퓨터에서 여러 명 한 번에 등록
  • 미리 받은 명단으로 사전 등록
Q6: 알림은 어떻게 받을 수 있나요?

A: 다양한 방법으로 실시간 알림을 받을 수 있습니다.

웹 푸시 알림:

  • 컴퓨터나 스마트폰 브라우저로 즉시 알림
  • 설정에서 "알림 허용" 한 번만 클릭
  • 인터넷 연결 시 실시간 수신
Q7: 방문자 데이터를 Excel로 내보낼 수 있나요?

A: 네, 언제든지 Excel 파일로 다운로드할 수 있습니다.

내보내기 기능:

  • 전체 데이터: 모든 방문자 기록
  • 기간별: 특정 날짜 범위 지정
  • 농장별: 여러 농장 중 선택
  • 항목별: 필요한 정보만 선택

활용 방법:

  • 정부 신고 및 보고서 작성
  • 방역 당국 자료 제출
  • 보험 처리 시 증빙 자료
  • 자체 분석 및 통계 작성

🔧기술적 문제

Q8: 인터넷 속도가 느린데 사용할 수 있나요?

A: 네, 낮은 속도에서도 원활하게 사용할 수 있도록 최적화되어 있습니다.

최소 요구사항:

  • 인터넷 속도: 1Mbps 이상
  • 데이터 사용량: 월 1GB 미만
  • 대부분의 농촌 인터넷 환경에서 사용 가능

최적화 기능:

  • 압축된 데이터 전송
  • 이미지 자동 최적화
  • 캐시 기능으로 빠른 로딩
Q9: 스마트폰이 오래되었는데 QR코드가 작동할까요?

A: 대부분의 스마트폰에서 사용 가능합니다.

지원 기기:

  • iPhone: iOS 12.0 이상 (iPhone 6 이상)
  • Android: 5.0 이상 (2014년 이후 기기)
  • 대부분의 QR코드 앱과 호환

문제 해결 방법:

  • 기본 카메라 앱 사용
  • QR코드 전용 앱 설치
  • 브라우저 업데이트
  • 카메라 렌즈 청소
Q10: 정전이나 인터넷 장애 시에는 어떻게 하나요?

A: 비상 상황에 대한 백업 방안을 제공합니다.

비상 대응책:

  • 종이 양식: 미리 준비된 방문자 기록지
  • 모바일 데이터: 스마트폰으로 임시 운영
  • 배터리 백업: UPS 장비 설치 권장
  • 복구 후 일괄 입력: 정상화 후 종이 기록 입력

데이터 보호:

  • 클라우드 자동 백업
  • 여러 서버에 분산 저장
  • 99.9% 가용성 보장
  • 24시간 모니터링

삭제 정책:

  • 복구 불가능한 완전 삭제
  • 삭제 확인서 발급

권한 관리:

Q18: 다크모드/라이트모드는 어떻게 바꾸나요?

A: 사이드바 하단에 있는 테마 토글 버튼을 클릭하면 됩니다.

다크모드 전환 방법:

  1. 사이드바 확인: 화면 왼쪽 사이드바 하단으로 스크롤
  2. 테마 버튼 찾기: 🌙 (달) 또는 ☀️ (해) 아이콘 버튼
  3. 클릭 전환: 버튼 클릭으로 다크모드/라이트모드 전환
  4. 즉시 적용: 변경 즉시 전체 화면에 적용

💡 팁:

  • 사이드바가 접혀있으면 먼저 펼쳐주세요
  • 모바일에서는 햄버거 메뉴(☰)를 눌러 사이드바 열기

Q22: 비밀번호를 잊어버렸을 때는 어떻게 하나요?

A: 로그인 페이지에서 "비밀번호 찾기" 기능을 사용하세요.

비밀번호 찾기 방법:

  1. 로그인 페이지: "비밀번호 찾기" 링크 클릭

  3. 인증 메일 발송: 비밀번호 재설정 링크가 포함된 메일 발송
  4. 링크 클릭: 메일의 링크를 클릭하여 새 비밀번호 설정
  5. 새 비밀번호 입력: 안전한 새 비밀번호로 변경
⚠️ 주의사항:
  • 이메일 주소가 정확한지 확인해주세요
  • 스팸 메일함도 확인해보세요
