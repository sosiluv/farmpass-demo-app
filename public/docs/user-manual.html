시스템 소개

🎯 농장 출입 관리 시스템이란?

농장 출입 관리 시스템(FarmPass)은 농장을 방문하는 모든 사람들의 출입을 체계적으로 관리하고 기록하는 웹 기반 시스템입니다.

✨ 주요 특징

  • 🔍방문자 추적: 누가, 언제, 왜 농장을 방문했는지 완벽 기록
  • 📱QR코드 지원: 스마트폰으로 간편한 출입 관리
  • 🔔실시간 알림: 방문자 등록 시 즉시 알림 수신
  • 📊통계 분석: 방문 패턴과 트렌드 분석
  • 🛡️보안 강화: 농장 보안과 방역 관리 지원
  • ☁️클라우드 기반: 언제 어디서나 접근 가능

🎉 시스템의 장점

  1. 방역 관리 향상: 모든 방문자 기록으로 질병 확산 방지
  2. 업무 효율성 증대: 수기 기록 대신 디지털 관리
  3. 법적 요구사항 준수: 농장 출입 기록 의무화 대응
  4. 데이터 기반 의사결정: 방문 통계로 농장 운영 최적화
  5. 비용 절감: 종이 기록부와 인력 비용 절약

시작하기

💻 시스템 요구사항

항목 요구사항
인터넷 연결 안정적인 인터넷 환경
웹 브라우저 Chrome, Firefox, Safari, Edge 최신 버전
모바일 Android 5.0+ 또는 iOS 12.0+
화면 해상도 최소 1024x768 (데스크톱), 360x640 (모바일)

🌐 시스템 접속 방법

  1. 웹 브라우저 실행
  2. 주소창에 입력: https://your-farm-system.com
  3. 첫 방문 시: 회원가입 진행
  4. 기존 사용자: 로그인

회원가입 및 로그인

👥 회원가입 절차

1단계: 회원가입 페이지 접속

  • 메인 페이지에서 "회원가입" 버튼 클릭
  • 또는 URL 직접 입력: https://your-farm-system.com/register

2단계: 기본 정보 입력

필드 예시 주의사항
이메일 주소 farmer@email.com 로그인 ID로 사용
비밀번호 Farm123!@# 최소 8자 이상, 대소문자, 숫자, 특수문자 포함
성명 홍길동 실제 이름 입력
연락처 010-1234-5678 하이픈(-) 포함

3단계: 회사/농장 정보 입력

  • 회사명: 농장명 또는 회사명 (예: 푸른농장)
  • 회사 주소: 농장 주소 (예: 경기도 김포시 농장로 123)
  • 업종: 농업 유형 선택 (축산업, 농업, 원예업 등)
  • 직책: 농장주, 관리자, 직원 등

4단계: 계정 인증

  1. 입력한 이메일로 인증 메일 발송
  2. 이메일 내 "계정 인증" 링크 클릭
  3. 인증 완료 후 로그인 가능

🔐 로그인 방법

  1. 로그인 페이지 접속: https://your-farm-system.com/login
  2. 이메일 주소 입력: 가입 시 사용한 이메일
  3. 비밀번호 입력: 설정한 비밀번호
  4. "로그인" 버튼 클릭

🔑 비밀번호 재설정

비밀번호를 잊어버린 경우:

  1. 로그인 페이지에서 "비밀번호를 잊으셨나요?" 클릭
  2. 가입한 이메일 주소 입력
  3. 이메일로 발송된 재설정 링크 클릭
  4. 새 비밀번호 설정
  5. 새 비밀번호로 로그인

농장 관리

🏡 농장 등록하기

로그인 후 첫 번째로 해야 할 일은 농장을 등록하는 것입니다.

1단계: 농장 추가

  1. 대시보드에서 "농장 관리" 메뉴 클릭
  2. "새 농장 추가" 버튼 클릭

2단계: 농장 정보 입력

필드 예시 설명
농장명 푸른목장 농장 이름
농장 주소 경기도 김포시 농장로 123 정확한 주소 입력
상세 주소 1동 축사 동/호수, 추가 위치 정보
농장 유형 축산업 (한우) 축산업, 농업, 원예업 등

3단계: 관리자 정보

  • 관리자명: 농장 현장 관리자 이름
  • 관리자 연락처: 비상 시 연락 가능한 번호
  • 추가 메모: 특별한 주의사항이나 정보

👥 농장 구성원 관리

여러 명이 한 농장을 관리하는 경우:

구성원 추가

  1. 농장 상세 페이지에서 "구성원 관리" 클릭
  2. "구성원 추가" 버튼 클릭
  3. 추가할 사람의 이메일 주소 입력
  4. 권한 레벨 선택

권한 레벨

  • 소유자: 모든 권한 (농장 삭제, 구성원 관리 등)
  • 관리자: 방문자 관리, 통계 조회
  • 조회자: 방문자 목록 조회만 가능

방문자 관리

👤 방문자 등록하기

방법: QR코드로 방문자 자가 등록

  1. QR코드 생성: 농장별 전용 QR코드 제공
  2. QR코드 배치: 농장 입구나 안내판에 부착
  3. 방문자 스캔: 방문자가 스마트폰으로 QR코드 스캔
  4. 정보 입력: 방문자가 직접 정보 입력
  5. 즉시 등록: 입력 완료 시 자동 등록

필수 입력 정보

  • 성명: 방문자 실명 (필수)
  • 연락처: 휴대폰 번호 (필수)
  • 주소: 방문자 거주지 주소 (필수)
  • 방문 목적: 방문 사유 (예: 사료 배송, 컨설팅, 견학 등)
  • 차량 번호: 차량 이용 시 번호판 정보
  • 소독 여부: 출입 전 소독 실시 여부

📋 방문자 목록 조회

기본 목록 보기

  1. "방문자 관리" 메뉴 클릭
  2. 등록된 방문자 목록 확인
  3. 각 방문자별 정보 확인

필터 및 검색

  • 기간 검색: 특정 날짜 범위의 방문자
  • 이름 검색: 방문자명으로 검색
  • 연락처 검색: 전화번호로 검색
  • 방문 목적: 목적별 필터링
  • 농장별: 여러 농장 관리 시 농장별 조회

📤 방문자 데이터 내보내기

Excel 파일로 내보내기

  1. "방문자 관리" 페이지에서 "내보내기" 버튼 클릭
  2. 내보내기 옵션 선택
  3. "다운로드" 클릭
  4. 컴퓨터에 파일 저장

활용 방법

  • 정부 신고: 농장 출입 기록 제출 시 활용
  • 방역 추적: 질병 발생 시 방문자 추적
  • 통계 분석: 외부 프로그램에서 데이터 분석

QR코드 시스템

📱 QR코드란?

QR코드는 스마트폰 카메라로 스캔하면 웹페이지로 바로 이동할 수 있는 2차원 바코드입니다.

🎯 QR코드 활용 방법

1단계: QR코드 생성

  1. "농장 관리" 메뉴에서 농장 선택
  2. "QR코드 생성" 버튼 클릭
  3. 농장별 전용 QR코드 자동 생성

2단계: QR코드 인쇄 및 설치

  1. 생성된 QR코드 "인쇄" 버튼 클릭
  2. A4 용지에 인쇄
  3. 농장 입구, 게이트, 안내판에 부착
  4. 비나 바람을 막을 수 있는 투명 케이스 사용 권장

QR코드 설치 위치

  • 🚪 농장 입구 게이트
  • 📋 방문자 대기실
  • 🏢 사무실 입구
  • 🚗 주차장 안내판

3단계: 방문자 안내

방문자에게 다음과 같이 안내:

  1. 스마트폰 카메라 앱 실행
  2. QR코드에 카메라 조준
  3. 화면에 나타나는 링크 터치
  4. 방문자 등록 페이지에서 정보 입력

📋 QR코드 방문자 등록 과정

방문자가 QR코드를 스캔하면:

  1. 농장 정보 확인: 어떤 농장인지 표시
  2. 방문자 정보 입력: 성명, 연락처, 주소 등
  3. 개인정보 동의: 수집 및 이용 동의
  4. 등록 완료: 즉시 시스템에 등록
  5. 알림 발송: 농장 관리자에게 실시간 알림

알림 기능

🔔 알림 시스템 개요

시스템에서 제공하는 실시간 알림으로 농장 상황을 즉시 파악할 수 있습니다.

📲 알림 설정하기

웹 푸시 알림 설정

  1. "설정" 메뉴 → "알림 설정" 클릭
  2. "웹 푸시 알림 허용" 버튼 클릭
  3. 브라우저 권한 요청 창에서 "허용" 클릭
  4. 알림 유형별 설정

알림 유형

  • 방문자 등록: 새 방문자 등록 시 알림
  • 긴급 알림: 시스템 문제나 보안 이벤트
  • 시스템 알림: 업데이트나 점검 공지

📢 받을 수 있는 알림 유형

방문자 관련 알림

  • 새 방문자 등록: "김철수님이 오후 2시에 푸른농장을 방문 등록했습니다"

시스템 알림

  • 시스템 점검: "오늘 밤 12시부터 2시간 시스템 점검 예정"
  • 업데이트 완료: "새로운 기능이 추가되었습니다"

관리자 기능

👨‍💼 관리자 계정이란?

관리자 계정은 시스템 전체를 관리할 수 있는 특별한 권한을 가진 계정입니다.

⚙️ 시스템 설정 관리

기본 설정

  1. "관리자" 메뉴 → "시스템 설정" 클릭
  2. 설정 가능한 항목 확인
  3. 필요한 설정 변경
  4. 변경사항 저장
설정 항목 설명
사이트명 시스템 제목 변경
사이트 설명 메인 페이지 설명 문구
로고 회사/농장 로고 업로드
언어 한국어, 영어 등 선택

보안 설정

  • 최대 로그인 시도: 계정 잠금 전 시도 횟수
  • 계정 잠금 시간: 잠금 해제까지 대기 시간
  • 비밀번호 정책: 최소 길이, 복잡성 요구사항

방문자 정책

  • 재방문 허용 간격: 동일 방문자의 재등록 가능 시간
  • 일일 최대 방문자: 하루 등록 가능한 최대 방문자 수
  • 데이터 보관 기간: 방문자 기록 보관 일수

📊 통계 및 분석

대시보드 통계

  1. "관리자" → "대시보드" 메뉴
  2. 확인 가능한 정보 검토

확인 가능한 정보

  • 오늘 방문자 수: 금일 등록된 방문자 현황
  • 주간 방문자 트렌드: 최근 7일간 방문자 변화
  • 월간 통계: 이번 달 총 방문자 수
  • 농장별 현황: 각 농장의 방문자 수 비교

📋 시스템 로그 조회

로그 유형

  1. "관리자" → "시스템 로그" 메뉴
  2. 로그 목록 확인

확인 가능한 로그

  • 사용자 활동: 사용자 로그인, 로그아웃, 활동 기록
  • 방문자 관리: 방문자 등록, 수정, 삭제 기록
  • 시스템 이벤트: 시스템 오류, 점검, 업데이트 기록
  • 보안 이벤트: 비정상 접근, 로그인 실패 기록, 알림 발송

로그 필터링

  • 날짜 범위: 특정 기간의 로그만 조회
  • 로그 레벨: 정보, 경고, 오류별 필터
  • 사용자별: 특정 사용자의 활동만 조회
  • 카테고리별: 농장, 방문자, 시스템별 분류

👥 사용자 관리

사용자 목록 조회

  1. "관리자" → "사용자 관리" 메뉴
  2. 전체 가입자 목록 확인
  3. 사용자별 정보 검토

사용자 계정 관리

  • 계정 활성화/비활성화: 문제 사용자 접근 차단
  • 권한 변경: 일반 사용자를 관리자로 승격
  • 계정 삭제: 필요 시 계정 완전 삭제

문제해결 및 지원

❓ 자주 묻는 질문 (FAQ)

Q1: 회원가입 후 이메일 인증이 오지 않아요

A: 다음을 확인해주세요:

  1. 스팸 메일함 확인
  2. 이메일 주소가 정확한지 확인
  3. 도메인 차단 설정 확인
  4. 10분 후에도 오지 않으면 "재발송" 버튼 클릭

Q2: 비밀번호를 잊어버렸어요

A:

  1. 로그인 페이지에서 "비밀번호를 잊으셨나요?" 클릭
  2. 가입한 이메일 주소 입력
  3. 이메일로 받은 링크를 통해 새 비밀번호 설정

Q3: QR코드가 스캔되지 않아요

A: 다음을 시도해보세요:

  1. 스마트폰 카메라 렌즈 청소
  2. QR코드가 훼손되지 않았는지 확인
  3. 충분한 조명 확보
  4. 카메라를 QR코드에서 적당한 거리로 조정
  5. 브라우저 업데이트 확인

Q4: 방문자 등록이 안 돼요

A: 확인사항:

  1. 모든 필수 항목이 입력되었는지 확인
  2. 연락처 형식이 올바른지 확인 (010-0000-0000)
  3. 개인정보 동의 체크박스 확인
  4. 인터넷 연결 상태 확인

Q5: 알림이 오지 않아요

A: 설정 확인:

  1. 브라우저 알림 권한 허용 여부
  2. 시스템 설정에서 알림 활성화 여부
  3. 스마트폰의 웹 브라우저 알림 설정
  4. 카카오톡 알림의 경우 연동 상태 확인
