📋 시작 전 체크리스트

✅ 준비물

  • ☐ 인터넷 연결된 컴퓨터 또는 스마트폰
  • ☐ 이메일 주소 (Gmail, Naver 등)
  • ☐ 휴대폰 번호
  • ☐ 농장 기본 정보 (농장명, 주소 등)

✅ 소요 시간

  • 회원가입: 3분
  • 농장 등록: 2분
  • 첫 방문자 등록: 2분

1
회원가입 3분

1️⃣ 시스템 접속

🌐 웹 브라우저 주소창에 입력
https://your-farm-system.com

📱 모바일에서도 동일한 주소 사용

2️⃣ 회원가입 정보 입력

항목 입력 예시 주의사항
이메일 admin@demo.com 로그인 ID로 사용
비밀번호 demo123!@# 8자 이상, 특수문자 포함
이름 홍길동 실제 이름
연락처 010-1234-5678 하이픈(-) 포함
농장명 푸른목장 대표 농장명

3️⃣ 이메일 인증

  1. 가입 완료 후 이메일 확인
  2. "계정 인증" 링크 클릭
  3. ✅ 인증 완료!
2
농장 등록 2분

1️⃣ 로그인 후 농장 추가

대시보드 → "농장 관리" → "새 농장 추가"

2️⃣ 농장 정보 입력

필수 정보 입력 예시
농장명 푸른목장 1농장
주소 경기도 김포시 농장로 123
농장 유형 축산업 (한우)
관리자명 홍길동
관리자 연락처 010-1234-5678

3️⃣ 저장 완료

✅ 농장 등록 완료!
3
첫 방문자 등록 2분

1️⃣ 방문자 등록

"방문자 관리" → "새 방문자 등록"

2️⃣ 방문자 정보 입력

필수 정보 입력 예시
방문자명 김철수
연락처 010-9876-5432
주소 서울시 강남구 테헤란로 456
방문 목적 사료 배송
방문 일시 오늘 오후 2시

3️⃣ 개인정보 동의 체크

  • [x] 개인정보 수집 및 이용 동의

4️⃣ 등록 완료

✅ 첫 방문자 등록 완료!
4
QR코드 설치 5분

1️⃣ QR코드 생성

"농장 관리" → 농장 선택 → "QR코드 생성"

2️⃣ QR코드 인쇄

  1. "인쇄" 버튼 클릭
  2. A4 용지에 인쇄
  3. 투명 파일에 넣어 보호

3️⃣ QR코드 설치 위치

🚪 농장 입구 게이트
📋 방문자 대기실
🏢 사무실 입구
🚗 주차장 안내판

4️⃣ 테스트

  1. 스마트폰으로 QR코드 스캔
  2. 방문자 등록 페이지 확인
  3. ✅ 설치 완료!
5
알림 설정 1분

1️⃣ 알림 권한 허용

"설정" → "알림 설정" → "웹 푸시 알림 허용"

2️⃣ 브라우저 권한 허용

브라우저 팝업에서 "허용" 클릭

3️⃣ 알림 유형 선택

  • [x] 방문자 등록 알림
  • [x] 긴급 알림
  • [x] 시스템 알림
✅ 실시간 알림 설정 완료!
6
테마 설정 30초

1️⃣ 사이드바 하단 확인

화면 왼쪽 사이드바 → 하단으로 스크롤 → 🌙/☀️ 버튼

2️⃣ 테마 전환

  • 라이트모드: ☀️ 버튼 클릭 (기본)
  • 다크모드: 🌙 버튼 클릭 (야간 작업에 유리)
✅ 테마 설정 완료!
7
데이터 내보내기 2분

1️⃣ 방문자 관리 페이지 접속

사이드바 → "방문자 관리" → "내보내기" 버튼 클릭

2️⃣ 내보내기 설정

  • 기간 선택: 전체 기간 또는 특정 기간 설정
  • 형식 선택: Excel(.xlsx) 또는 CSV(.csv)
  • 필터 적용: 필요한 경우 특정 조건으로 필터링

3️⃣ 파일 다운로드

설정 완료 후 "다운로드" 버튼을 클릭하면 파일이 자동으로 다운로드됩니다.

✅ 데이터 내보내기 완료!
8
모바일 최적화 3분

1️⃣ PWA 설치 (권장)

브라우저 주소창 → "설치" 아이콘 클릭 → "설치" 확인

2️⃣ 모바일 최적화 확인

  • 터치 최적화: 모든 버튼이 터치하기 쉽게 설계
  • 반응형 디자인: 화면 크기에 자동으로 최적화
  • 오프라인 지원: 인터넷 없이도 기본 기능 사용
  • 푸시 알림: 실시간 알림 수신

3️⃣ 모바일 사용 팁

  • 가로 모드에서 더 넓은 화면 활용
  • PWA 설치로 앱처럼 빠른 접근
  • 홈화면 바로가기로 즉시 접속
✅ 모바일 최적화 완료!

🤔 자주 발생하는 문제

Q: QR코드를 스캔했는데 아무것도 안 나와요
A: 다음을 확인해주세요
  • 스마트폰 카메라 앱으로 스캔
  • QR코드가 선명한지 확인
  • 충분한 조명 확보
Q: 알림이 오지 않아요
A: 설정을 확인해주세요
  • 브라우저 알림 권한 허용
  • 시스템 설정에서 알림 활성화
  • 스마트폰 알림 설정 확인
Q: 로그인이 안 돼요
A: 다음을 시도해보세요
  • 이메일 주소와 비밀번호 재확인
  • Caps Lock 키 확인
  • 브라우저 새로고침
